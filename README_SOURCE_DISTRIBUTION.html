The source distribution includes the Java source files, excepting "Rest.java", and it incudes Python code for generating "Rest.java". Information for generating Indivo server client software, regardless of computer language, is contained in api.xml. indivo_client_java source includes code for generating Rest.java from api.xml, instead of including Rest.java directly. Generate Rest.java

Pre-requisite: python 2.6 or equivalent.

  • cd <indivo_client_java-directory>/client_codegen/bin
  • ./genCodeFromPython.sh 1> temp1  When complete, the last line of temp1 should show something like: "done. 140 skeleton calls processed. 124 Java methods generated." This may vary depending on the version you downloaded.
  • ./compile.sh  This might produce warnings. There should be no errors. This will create the file <indivo_client_java-directory>/src/main/java/org/indivo/client/Rest.java.
  • ./javadoc.sh 1> temp1  When complete, temp1 should show no errors, and <indivo_client_java-directory>/javadoc will be populated.

Use client API
